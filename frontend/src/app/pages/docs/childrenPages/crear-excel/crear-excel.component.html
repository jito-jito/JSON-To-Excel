Crear un excel

Antes de comenzar, prepara tus datos en un archivo JSON siguiendo el formato especificado acontinuación.

Formato del JSON

El formato del JSON debe seguir la estructura definida para la generación del Excel. Aquí hay una descripción de la función que tiene cada propiedad:

  • tableType: string (obligatorio) : Indica el tipo de disposición de la tabla, actualmente solo compatible con el tipo: 'byRow'.
  • sheetName: string (opcional) : Especifica el nombre de la hoja de Excel que se utilizará para crear la tabla, por defecto el valor es 'hoja 1'.
  • title: string (opcional) : Define el título principal de la tabla en el Excel, por defecto el valor es 'titulo de la tabla'.
  • headerColumns: string[] (obligatorio) : Lista las columnas de encabezado en la tabla, los valores ingresados aquí deben coincidir con el nombre de las propiedades indicadas en los datos a cargar.
  • data: {{ '{' }}[key: string]: string {{ '}' }} (obligatorio) : Contiene los datos de la tabla en un formato de matriz de objetos, el nombre de la propiedad debe ser identico al nombre agregado en 'headerColumns' ya que será utilizado para definir en que columna de la tabla ingresaremos el valor.
    si agregas una propiedad con un nombre diferente este valor será ignorado en el registro de la tabla.
