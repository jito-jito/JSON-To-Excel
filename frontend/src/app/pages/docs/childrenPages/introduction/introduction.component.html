Introducción

¡Bienvenido a la documentación oficial de la aplicación Excel con JSON!

Descripción

Esta aplicación cumple el propósito de generar una tabla en un documento Excel con datos de un archivo JSON, está pensada para hacerle la vida sencilla al desarrollador, no te ha pasado que necesitas entregar la información de respuesta de una API o del archivo de logs de forma ordenada a tu jefe o cliente?, pues aquí tienes la solución!, solo necesitas que tu JSON cumpla con la estructura definida y YO, tu aplicación favorita hará el resto.

Versiones y Actualizaciones

La aplicación se encuentra en su versión beta, actualmente cumple con un único formato disponible para generar una tabla de Excel, si deseas ver las últimas actualizaciones, notas de versión o ojear el código te invito a ver el repo en GitHub.

Objetivos de la Documentación

Esta documentación tiene como objetivo proporcionar información detallada y guías para darle el formato al archivo JSON que Te ayudará a crear la plantilla excel que necesitas.

Público Objetivo

Como ya mencioné, mi objetivo es que la herramienta ayude a desarrolladores ahorrándoles tiempo en crear un archivo con los datos que necesitas exponer de manera ordenada, ya sea una API, un archivo de logs o quien sabe. De forma sencilla, sin tantas configuraciones, principalmente agregando los datos que necesitas visualizar...
